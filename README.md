# 🚗 Car Price Prediction Analysis
An end-to-end Machine Learning project utilizing Regression and KNN Classification to estimate vehicle market value based on features like year, mileage, and engine size.

## 📊 Project Highlights
* **Data Cleaning:** Extensive outlier removal and feature scaling.
* **Regression:** Achieved an R² of 0.71 predicting continuous market prices.
* **Classification:** Optimized a KNN model via Grid Search to achieve **87% accuracy** in market segment categorization.
* **Optimization:** Used 5-Fold Cross-Validation and Manhattan distance metrics.

## 🛠️ Tech Stack
* Python (Pandas, NumPy, Scikit-Learn)
* Visualization (Matplotlib, Seaborn)

## 📁 Repository Structure
* `Assignment.ipynb`: The main project notebook containing EDA, Modeling, and Analysis.
* `car_data.csv`: (Optional) The raw dataset used for the project.
